- **Pipeline Flow** — step-by-step from code push to live deployment
- **Popular CI/CD Tools** — GitHub Actions, Jenkins, GitLab CI, CircleCI, AWS CodePipeline and more
- **GitHub Actions YAML Keywords** — on, jobs, runs-on, steps, uses, run, needs, secrets
- **Delivery vs Deployment** — comparison table of approval, speed, risk, and usage
